# 配線図（desk_fan_controller.ino 対応）

この配線図は、Arduino UNO R3 と `desk_fan_controller.ino` のLCD未使用版に合わせたものです。

## 1. ピン対応表

| 機能 | 部品 | Arduinoピン | 備考 |
|:--|:--|:--|:--|
| 人感センサー入力 | HC-SR501 OUT | D2 | VCC=5V, GND=GND |
| ボタン入力 | タクトスイッチ | D3 | INPUT_PULLUP（もう片側はGND） |
| モーター有効化 | モーター駆動回路 | D4 | トランジスタ回路へ |
| モーターPWM | モーター駆動回路 | D5 | 1kΩ経由でトランジスタBase |
| 状態LED | 5mm LED | D6 | 220Ω直列でGNDへ |

---

## 2. 全体配線図（テキスト）

```text
[Arduino UNO R3]
  5V  ---------------------------> +5V レール
  GND ---------------------------> GND レール

  D2  ---------------------------> HC-SR501 OUT
  5V  ---------------------------> HC-SR501 VCC
  GND ---------------------------> HC-SR501 GND

  D3  ---------------------------> タクトスイッチ片側
  GND ---------------------------> タクトスイッチ反対側

  D6  ---- 220Ω -----> LEDアノード(+)
  GND ---------------------------> LEDカソード(-)

  D5  ---- 1kΩ ------> NPNトランジスタ Base
  GND ---------------------------> NPNトランジスタ Emitter
  NPNトランジスタ Collector -----> モーター(-)
  +5Vレール ---------------------> モーター(+)

  ダイオード(1N4007)をモーターに並列:
    カソード -> モーター(+)
    アノード -> モーター(-)

  D4  ---------------------------> モーター有効化制御線（回路構成に応じて使用）
```

---

## 3. Fritzing風の穴番号つき配置図（ブレッドボード）

前提: 標準的なハーフサイズブレッドボード（a-e / f-j の中央溝あり）を想定。

### 3-1. 主要部品の配置（穴番号指定）

| 部品 | 配置位置（穴番号） | 補足 |
|:--|:--|:--|
| タクトスイッチ | `e20/e22` と `f20/f22` にまたぐ | 中央溝をまたぐ向きで配置 |
| LED（状態表示） | アノード=`j25` / カソード=`j27` | 直列抵抗とセットで接続 |
| LED直列抵抗 220Ω | `f25` - `f30` | `f25`側をLEDアノード系へ |
| NPNトランジスタ | `e35`(B) / `e36`(C) / `e37`(E) | 型番で足順が異なるため要確認 |
| ベース抵抗 1kΩ | `a35` - `a40` | `a40`側をD5へ |
| モーター端子(−) | `j36` | トランジスタC側へ |
| モーター端子(+) | `+レール` | 5V給電 |
| ダイオード 1N4007 | `j36` - `+レール` | カソードを+側へ |

### 3-2. ジャンパ線の例（穴番号付き）

| 信号 | 配線例 |
|:--|:--|
| D3（ボタン入力） | Arduino D3 → `a20` |
| GND（ボタン側） | GNDレール → `j22` |
| D6（LED制御） | Arduino D6 → `a30` → 抵抗(`a30-f30`) → LEDアノード系(`f25-j25`) |
| LED GND | LEDカソード `j27` → GNDレール |
| D5（PWM） | Arduino D5 → `a40` → 1kΩ → `a35` → トランジスタB(`e35`) |
| トランジスタE | `e37` → GNDレール |
| トランジスタC | `e36` ↔ `j36`（モーター−） |
| モーター+ | モーター+ → +5Vレール |

### 3-3. 未使用ピン（今回の簡易構成）

- D7〜D12 は未使用（LCDを使わない）
- A0〜A5 も未使用

---

## 4. 配線チェックリスト

- D2/D3/D4/D5/D6 以外に不要配線が残っていない
- モーターを Arduino ピンへ直結していない
- トランジスタとダイオードの向きが正しい
- すべてのGNDが共通になっている
