# 回路の組み立て手順（チュートリアル形式）

この手順は ELEGOO のチュートリアル（ページ3 風）に合わせたブレッドボード組み立てと初回チェック手順です。作業前に配線図（`src/diagrams/wiring_elegoo_page3.svg`）を参照してください。

## 準備（安全）
- 作業は必ず電源を切った状態で行う。
- モーターは初回は未接続でテストする（誤配線による破損防止）。
- 今回は Arduino の 5V のみでモーターを駆動する想定です（モーターの + は Arduino 5V に接続します）。
- 外部電源を使う場合は、Arduino の GND と外部電源の GND を必ず共通化してください。
 - 必要部品: Arduino (Uno/Nano)、DHT11 モジュール、3x 押しボタン、NPN トランジスタ（低側スイッチ用）、3–6V DC モーター（共通部品）、フライバックダイオード(1N400x)、1kΩ、100kΩ、220–330Ω（LED 用）、ジャンパー、ブレッドボード。


## 部品のピン配置（参照）
- DHT11: VCC, DATA, GND
- Buttons: 片側をデジタルピン（D3,D4,D5）へ、もう片側を GND へ接続（`INPUT_PULLUP` 前提）
- トランジスタ: Base、Collector（モータ-）、Emitter（GND）
- LEDs: White→D13、Green→A0、Red→A1（各 220–330Ω）


## 組み立て手順（順を追って）

1. Arduino をブレッドボードに配置する
   - Arduino をブレッドボード近傍に置き、5V と GND のレールが使えるようにする。

2. 電源ラインを準備する
   - ブレッドボードの + レールを Arduino の 5V に、- レールを Arduino の GND に接続する。
   - 外部モーター電源を使う場合はまだ+側を接続しないでおく（後で接続）。

3. DHT11 を接続する
   - DHT11 の VCC → Arduino 5V
   - DHT11 の DATA → Arduino D2
   - DHT11 の GND → Arduino GND
   - 必要なら DATA と 5V 間に 4.7k–10k のプルアップを入れる

4. ボタンを接続する（Power / WEAK / STRONG）
   - 各ボタンの一端をそれぞれ Arduino の D3, D4, D5 に接続
   - ボタンのもう一端は GND に接続
   - Arduino ソフトは `pinMode(pin, INPUT_PULLUP)` を使う想定

5. トランジスタ とモーターの配線
   - トランジスタをブレッドボードに差す
   - Arduino D6 → 1kΩ（ベース抵抗）→ トランジスタ Base
   - トランジスタ Emitter → Arduino GND（共通）
   - トランジスタ Collector → モーターのマイナス端
   - モーターのプラス端は Arduino の 5V（今回の想定）。想定するモーター定格は 3–6V の小型 DC モーターです。5V 給電は定格内ですが、定格電流・始動電流を必ず確認してください。下記の「電源注意」を必ず確認してください。
   - モーター両端にフライバックダイオードを並列で接続（カソード→+V、アノード→モーター-）
   - Base と GND 間に 100kΩ のプルダウンを接続（ベースの浮遊防止）

6. LED を接続する
   - White LED（電源表示）: D13 → 220Ω → LED アノード、LED カソード → GND
   - Green LED（弱）: A0 → 220Ω → LED アノード、LED カソード → GND
   - Red LED（強）: A1 → 220Ω → LED アノード、LED カソード → GND

7. 配線の最終確認（電源投入前）
   - 各配線が間違っていないかチェック（特にトランジスタの B/C/E、モーターの +/−）
   - フライバックダイオードの方向を確認（カソード→+V）
   - ベースに直列抵抗 1kΩ とベース→GND に 100kΩ のプルダウンがあることを確認
   - 外部電源を使用する場合は Arduino と外部電源の GND を共通にする（今回の設定では Arduino 5V のみ）。

8. 初回電源投入と動作確認（モーター未接続）
   - モーターの + を接続しない状態で Arduino を USB 給電する
   - シリアルモニタ（9600bps）を開き、DHT11 の読み取りログが出るか確認する
   - ボタンを押してそれぞれの入力が検出されるか確認（押下で LOW）
   - LED が想定どおり点灯するか確認

9. モーター接続と最終確認
   - DHT11/ボタン/LED が正常ならモーターの + を Arduino の 5V に接続（今回の想定）。
   - 短時間（数秒）だけ PWM を使ってモーターを回し、トランジスタ の発熱や異音がないか確認する
   - 【電源注意】Arduino 5V 給電は USB/レギュレータの供給に制約があります。モーターの定格電流・始動電流が大きい場合は外部電源を使用してください。心配な場合はモーター仕様（型番・定格電流）を教えてください。
   - 異常（過熱・煙・異臭・不安定動作）を感じたら直ちに電源を切る

## トラブルシュート（簡潔）
- 白・緑 LED が常時点灯する
  - ボタン配線が逆（INPUT_PULLUP 前提）でないか確認
  - LED の配線が VCC 側に直結されていないか確認
 - モーターが止まらない（ベース を LOW にしても止まらない）
   - トランジスタ が壊れている可能性（導通をテスターで確認）
   - モーターが 5V レールに直接接続されていないか確認
   - トランジスタ のピン配置が誤っていないか（B/C/E）を確認
- DHT11 が読み取れない
  - DATA ラインにプルアップ 4.7k–10k を入れて再試行
  - 配線（VCC/GND/DATA）を再確認

## トランジスタ（NPN）で代用する場合の配線例
以下はトランジスタ（NPN）を低側スイッチとして使う場合の手順と注意点です。モーターの消費電流が大きい場合はトランジスタの損失・発熱に注意してください。

- 推奨接続（低側スイッチ）:
   - Arduino D6 --[Base 抵抗 Rb]--> Base
   - Collector --> モーターの -
   - モーターの + --> 外部電源 +V
   - Emitter --> 共通 GND
   - モーターに並列でフライバックダイオード（カソード→+V、アノード→モーター -）を必ず入れる
   - Base と GND の間に 100kΩ 程度のプルダウンを付けると確実に OFF になる

- 手持ち抵抗に合わせた Base 抵抗の目安（今回の手持ち: 1k, 2k, 5k, 100k）:
   - 推奨: `1kΩ`（Base 電流 ≒ (5V−0.7V)/1k ≒ 4.3mA）。小〜中程度の電流のモーターで安定しやすい。
   - 5kΩ はベース電流が小さく飽和しにくいため、モーター電流が小さい場面でのみ検討。
   - 100kΩ は大きすぎて動作しない可能性が高い（ベース電流不足）。

- 飽和と安全性の目安:
   - BJT を飽和させるには一般に Ib ≈ Ic/10 を目安とします。例: モーターが 300mA を消費する場合、理想的には Ib ≈ 30mA が必要（手持ちの 1kΩ では不足する）。
   - よって、モーターが 200–300mA を超える場合はトランジスタでは発熱・飽和不足のリスクがあるため、パワートランジスタや低損失のスイッチング素子の使用を推奨します。

- 実務的手順（トランジスタ版）:
   1. 駆動回路をトランジスタに置き換え、コレクタをモーター−、エミッタを GND に接続する。
   2. Arduino D6 とトランジスタ Base の間に `1kΩ` を接続する（手持ちがあればまず 1k を使用）。
   3. モーター両端にフライバックダイオードを配置する（方向に注意）。
   4. 外部電源の GND を Arduino GND と共通化する。
   5. 初回は短時間（1〜2 秒）だけ PWM を入れてトランジスタの発熱を確認する。温度が高すぎる場合は電源を切る。

- 推奨パーツの指針:
   - モーター電流が小（≲100mA）: 小信号トランジスタ（2N2222 等）＋1kΩ で試せる。
   - モーター電流が中〜大（>100–200mA）: パワートランジスタ（例: TIP31 系）または低損失のスイッチング素子を使用すること。
   - Darlington（TIP120 等）は使えるが飽和電圧が高く、発熱と電圧降下が増えるため注意。

注意: 回路変更後は必ず短時間のテストと発熱確認を行い、必要なら放熱対策（ヒートシンク）を施してください。
